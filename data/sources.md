# Data Sources

## WHOI-Plankton (2014 subset)

- Dataset: WHOI-Plankton (Imaging FlowCytobot / IFCB)
- Year downloaded: 2014
- Download method: R + iRfcb (`ifcb_download_whoi_plankton`)
- Local storage (not committed): `data/whoi_plankton_raw/`
- Curated subset (phytoplankton only, not committed): `data/images/`

Notes:
- This repository does not include raw images due to size and licensing considerations.
- Only metadata format examples are included (`data/sample_metadata.csv`).
